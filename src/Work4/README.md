Phong与Blinn-Phong在高光区域边缘的视觉表现差异：


Phong 模型：高光边缘锐利、生硬，高光区域呈 “尖角状”，当观察方向与表面夹角较大（大入射角）时，高光容易出现断裂、消失的现象，过渡不自然。


Blinn-Phong 模型：高光边缘柔和、圆润，高光区域呈平滑过渡的 “光晕状”，即使在大入射角下，高光也能保持完整且自然的过渡，视觉效果更接近真实材质。
